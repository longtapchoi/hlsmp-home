# HLSMP-Home

Plugin nhà cho HLSMP Server — convert từ DonutSMP-Homes by Duuchniuk.

## Lệnh
- `/sethome <tên>` — Lưu vị trí làm nhà (tối đa 5)
- `/home [tên]` — Mở GUI hoặc teleport thẳng
- `/delhome <tên>` — Xóa nhà

## Tích hợp HLSMP-Teams
GUI `/home` slot 10 và 19 kết nối với HLSMP-Teams:
- Slot 10 (WHITE_BANNER) — click teleport Team Home
- Slot 19 — hiển thị trạng thái động:
  - Đỏ: chưa gia nhập Team
  - Đỏ: Team chưa đặt Home
  - Xanh: click để tele Team Home
